## Lessons Learned (12:53)

### preview-nginx Verzeichnis-Konvention
- **Problem:** Deploy schreibt nach `/opt/sites/preview-{site}/` (Prefix), aber `preview-nginx` liest aus `/opt/sites/{site}-preview/` (Suffix)
- **Fix:** Symlink: `ln -s /opt/sites/preview-{site} /opt/sites/{site}-preview` auf minicon-web
- **Betroffen:** Jede neue Preview-Site bis Symlink gesetzt ist
- **Langfristig:** nginx-Config auf `root /opt/sites/preview-$site;` umstellen (TODO)
- **Aktuell gefixt für:** `tv-dahn-preview -> preview-tv-dahn`

### Confluence task-list Macro
- **Problem:** `ac:structured-macro ac:name="task-list"` wird im neuen Confluence-Editor nicht supported
- **Fix:** Ersetzen durch natives `<ac:task-list><ac:task><ac:task-id>N</ac:task-id>...</ac:task></ac:task-list>` Format
- **Methode:** Confluence REST API `PUT /wiki/rest/api/content/{id}` mit Regex-Replace auf dem Storage-Body
